# Cinema Ticket

Working implementation of a Ticket Service.

## Scripts

Install dependencies

```powershell
npm install
```

Test the solution

```powershell
npm run test
```

Lint the solution

```powershell
npm run lint
```

Fix linting errors

```powershell
npm run lint:fix
```
